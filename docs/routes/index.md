# 推荐阅读路线

## 推荐教程路线

如果你希望这个仓库更像一套教程来读，而不是一篇长说明，建议按目标选择：

### 路线 1：第一次接触 ECC

1. 先看本页的“核心概念”。
2. 再看 [`ECC-COMMANDS-ZH.md`](../commands/index.md) 了解最常用命令。
3. 再看 [`ECC-NAVIGATION-ZH.md`](../upstream-navigation/index.md) 建立整个上游副本的目录地图。
4. 最后进入 [`everything-claude-code/README.md`](https://github.com/affaan-m/ECC/blob/main/README.md) 对照原始定义。

### 路线 2：想尽快学会怎么用

1. [`ECC-COMMANDS-ZH.md`](../commands/index.md)
2. 本页的“实际工作流示例”
3. [`everything-claude-code/COMMANDS-QUICK-REF.md`](https://github.com/affaan-m/ECC/blob/main/COMMANDS-QUICK-REF.md)
4. [`everything-claude-code/the-shortform-guide.md`](https://github.com/affaan-m/ECC/blob/main/the-shortform-guide.md)
